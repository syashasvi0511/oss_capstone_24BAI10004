# The Open Source Audit – Python

## 📌 Project Overview
This project is part of the Open Source Software course. It presents a detailed audit of Python as an open-source software, including its origin, licensing, ecosystem, and real-world usage in Linux environments.

## 🎯 Objective
The objective of this project is to understand:
- Open-source philosophy
- Licensing models
- Role of Python in the FOSS ecosystem
- Practical interaction with Linux systems
- Shell scripting fundamentals

## 🐍 Chosen Software
**Python**

Python is a widely used open-source programming language known for its simplicity, readability, and versatility in fields like:
- Web Development
- Artificial Intelligence
- Data Science
- Automation

## ⚙️ Features of the Project
- Detailed analysis of Python’s origin and philosophy
- Study of open-source licenses (GPL vs MIT)
- Linux-based installation and system interaction
- Exploration of Python ecosystem and dependencies
- Comparison between open-source and proprietary software
- Implementation of 5 shell scripts:
  - System Identity Report
  - Package Inspector
  - Disk & Permission Auditor
  - Log File Analyzer
  - Open Source Manifesto Generator

## 🖥️ Technologies Used
- Python
- Linux (Ubuntu)
- Shell Scripting (Bash)
- Git & GitHub
