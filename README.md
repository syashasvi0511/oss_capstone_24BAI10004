# The Open Source Audit – Python

## 📌 Project Overview
As an assignment in the Open Source Software course, this project provides a thorough audit of Python from its birth to today as an open source application, with regard to licensing, ecosystems and real world uses of Python in Linux based environments.

## 🎯 Objective
The main goal of this project is to explore how open-source software actually works in the real world. Instead of just learning theory, this project focuses on understanding Python from different angles—its background, how it is used in Linux systems, and why it is so popular among developers. It also helped me gain hands-on experience with shell scripting and gave me a better idea of how open-source communities collaborate and contribute.

## 🐍 Chosen Software
**Python**

For this project, I chose Python because it is one of the most beginner-friendly and widely used programming languages today. What makes Python interesting is how simple and readable it is, even for someone who is just starting out. At the same time, it is powerful enough to be used in advanced areas like artificial intelligence, web development, and data analysis. Its open-source nature and strong community support make it a perfect example to study in this audit.

## ⚙️ Features of the Project
This project covers both theoretical understanding and practical implementation. It includes a detailed study of Python’s background, its licensing, and its role in the open-source ecosystem. Along with that, I worked on Linux to see how Python interacts with the system in real life. One of the most interesting parts was writing shell scripts, which helped me understand how automation and system-level tasks are handled. Overall, the project combines research, analysis, and hands-on learning to give a complete view of open-source software.
